## 機能紹介
- 新規会員登録・ログイン機能
- ログイン時の商品の出品、商品の編集機能
- ログインしていない方でも商品の一覧、詳細を閲覧可能です。
- 住所、クレジットカード登録後商品の購入ができます。
- 自分が出品した商品をマイページで確認できます。

## 実装内容の紹介
- 商品情報編集機能
- 商品詳細表示機能
- 商品削除機能
- 商品出品機能
- ユーザー新規登録、ログイン機能
- カテゴリー機能
- カテゴリ別商品検索機能
- 商品検索機能
- 商品購入機能
- 商品一覧表示機能
- マイページ機能
- pay.jpのapiを用いたクレジットカード登録・削除機能 & 商品購入機能
